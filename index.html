BYE NY is an interactive infographic application that shows all flights departing from the three major airports in and around New York; Newark Int’l Airport, LaGuardia Airport and JFK Airport. Flight colors are based on destination country and are displayed in 15 minutes “bursts”.

Technologies: The application is built with Ruby and a Rails3 framework. The map was created using SVG (Scalable Vector Graph) and the flight lines were created using a d3.js canvas and data visualization. Tested with cucumber/capybara and rspec. Front-end was built with CSS3 and JavaScript.


Pool Party was created to help alleviate the insecurities and fear of carpooling with strangers. The app allow users to create a "trip" that would be visible to other users. Users would then be able to view profiles and give rating feedback/rating to other users. This helps save money and the planet!

Technologies: This app was built with Ruby on a Rails4 framework. Utilizes the latest of HTML5 and CSS3 technologies to create a welcoming user experience for the users. All users are requried to sign up with proper credentials to help ensure the safety of the users through devise authentication. Tested with rspec.


Heart Beat Ink is a website built for Andres Moran who is an aspiring tattoo artist and a friend. What started out as a hobby, quickly turned in to a passionate career. When you love what you do, it no longer becomes work, it becomes life.

Technologies: This website is built with Ruby on a Rails3 framework with scalability in-mind. As his business will inevitably grow, the website was designed to dynamically scale with his business.The website uses great CSS3 features along with a few JavaScript libraries to make navigating for the users a breeze. The site also has a complete admin access views taking full advantage of the MVC framework.
