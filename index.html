Recent Projects

BYE NY is an interactive infographic application that shows all flights departing from the three major airports in and around New York; Newark Int’l Airport, LaGuardia Airport and JFK Airport. Flight colors are based on destination country and are displayed in 15 minutes “bursts”.
The application is built with Ruby and a Rails3 framework. The map was created using SVG (Scalable Vector Graph) and the flight lines were created using a d3.js canvas and data visualization. The data was taken from an API provided by www.flightstats.com which was then parsed to show only departing flights and relavant information. Front-end was built with CSS3 and JavaScript.
